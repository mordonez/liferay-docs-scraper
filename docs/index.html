Skill-first workflow

Ask better questions against local sources.

Adds a skill

Installs `liferay-expert` into your project so your assistant has a focused Liferay research workflow.

Uses local files

Reads a Markdown corpus from your machine instead of relying on model memory alone.

Keeps answers cited

Uses source metadata so answers can point back to the URLs behind the local files.

Quickstart

Install the skill and verify the corpus.

uvx --from crawl4ai crawl4ai-setup
uvx liferay-docs-scraper
npx skills add mordonez/liferay-docs-scraper --skill liferay-expert -a claude-code
uvx --from liferay-docs-scraper liferay-docs-scraper-doctor

Why it matters

Built for technical answers that age well.

  • No third-party documentation content bundled in the package.
  • No vector database or embedding setup required.
  • One shared docs directory for every local project.
  • Freshness checks and reports through the doctor command.
