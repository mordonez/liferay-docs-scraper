Team-ready workflow

Give your assistant the docs before the question.

Adds a skill

Installs `liferay-expert` into your project so Claude Code knows how to search, read, and cite Liferay docs.

Uses local files

Builds one shared local context library for every project on your machine.

Keeps answers cited

Keeps source metadata with every page so answers can point back to the original Liferay URLs.

Quickstart

Build the context library and verify the skill.

uvx --from crawl4ai crawl4ai-setup
uvx liferay-context-builder
npx skills add mordonez/liferay-context-builder --skill liferay-expert -a claude-code
uvx --from liferay-context-builder liferay-context-builder-doctor

Previously published as liferay-docs-scraper. New installs should use liferay-context-builder.

Why it matters

Built for technical answers that age well.

  • No third-party documentation content bundled in the package.
  • No vector database or embedding setup required.
  • One shared docs directory for every local project.
  • Freshness checks and reports through the doctor command.
